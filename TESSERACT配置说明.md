# Tesseract OCR 配置说明

## 概述

项目已改用 **Tesseract OCR** 替代 ML Kit，支持离线识别日语和英语文字。

## 语言数据

由于语言数据文件较大（日语约35MB），无法直接放入APK中。应用采用**运行时下载**方案。

### 可用语言

| 语言 | 代码 | 大小 | 说明 |
|------|------|------|------|
| 英语 | eng | 约4MB | 基础拉丁字母识别 |
| 日语 | jpn | 约35MB | 日语横排文字 |
| 日语竖排 | jpn_vert | 约35MB | 日语竖排文字 |

## 使用步骤

### 1. 首次使用

1. 打开应用
2. 点击 **"语言数据下载"**
3. 下载所需的语言数据（至少下载英语）
4. 返回主界面，点击 **"启动翻译"**

### 2. 下载语言数据

- 点击 **"下载"** 按钮开始下载
- 下载进度会实时显示
- 下载完成后即可使用对应语言的OCR识别

### 3. 删除语言数据

- 点击 **"删除"** 按钮可删除已下载的语言
- 删除后可重新下载

## 存储空间

- 语言数据存储在应用私有目录
- 卸载应用时会自动清除
- 可在 **"语言数据下载"** 页面查看已用空间

## 网络要求

- 下载语言数据需要网络连接
- OCR识别完全离线，无需网络

## 注意事项

1. **首次使用必须下载语言数据**，否则OCR无法工作
2. 建议WiFi环境下下载日语数据（35MB较大）
3. 如果只识别英语，只需下载英语数据（4MB）
4. 下载失败可重试

## 技术说明

- 语言数据来源：https://github.com/tesseract-ocr/tessdata
- 使用 Tesseract 4.x 引擎
- 支持 Android 7.0+ (API 24+)
